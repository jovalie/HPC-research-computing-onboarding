# CARC Account

In this session, we will be working with you to make sure you have an account on the Laguna cluster.

1. Open [https://hpcaccount.usc.edu](https://hpcaccount.usc.edu)
2. Choose your school (It will bring you to your school's SSO)
3. Check if you are assigned to the Project named "CMC_Onboarding"
4. If you have any trouble, let us know.

For additional information, check out CARC's Getting Started guide:
[https://uschpc.github.io/regional-computing-website/user-guides/get-started-laguna.html](https://uschpc.github.io/regional-computing-website/user-guides/get-started-laguna.html)
